If/Else For While Função

If / Else (10 exemplos)

1. Simples
if x > 0:
    print("Positivo")
2. If-Else
if x > 0:
    print("Ok")
else:
    print("Erro")
3. Elif
if x > 10:
    print("Grande")
elif x > 5:
    print("Médio")
else:
    print("Pequeno")
4. Comparação
if x == 10:
    print("Igual a 10")
5. Diferença
if x != 0:
    print("Não é zero")
6. AND
if x > 0 and x < 10:
    print("Entre 0 e 10")
7. OR
if x == 1 or x == 2:
    print("1 ou 2")
8. In List
if x in [1, 2]:
    print("Na lista")
9. Not
if not x:
    print("Vazio/Zero")
10. isinstance
if isinstance(x, int):
    print("Inteiro")

For Loop (10 exemplos)

1. Range
for i in range(5):
    print(i)
2. String
for c in "Py":
    print(c)
3. Lista
for n in [1, 2]:
    print(n)
4. Step
for i in range(0, 10, 2):
    print(i)
5. Reverse
for i in reversed(range(3)):
    print(i)
6. Enumerate
for i, v in enumerate(['a', 'b']):
    print(i, v)
7. Dict keys
for k in {'a': 1}:
    print(k)
8. Tuple
for v in (1, 2):
    print(v)
9. Zip
for a, b in zip([1], [2]):
    print(a, b)
10. List Comp
[x*2 for x in range(3)]

While Loop (10 exemplos)

1. Simples
while x > 0:
    x -= 1
2. Break
while True:
    break
3. Contagem
while x < 5:
    x += 1
4. Decremento
while x > 0:
    x -= 1
5. Flag
run = True
while run:
    run = False
6. Pop list
while lista:
    lista.pop()
7. Cond OR
while x < 5 or y < 5:
    x += 1
8. Print
while x > 0:
    print(x)
    x -= 1
9. Pass
while False:
    pass
10. Continue
while x > 0:
    x -= 1
    continue

Funções (10 exemplos)

1. Vazia
def f():
    pass
2. Retorno
def f():
    return 1
3. Argumento
def f(a):
    return a
4. Padrão
def f(a=1):
    return a
5. Lambda
f = lambda x: x * 2
6. Docstring
def f():
    """Doc"""
7. Args
def f(*args):
    pass
8. Kwargs
def f(**kwargs):
    pass
9. Global
def f():
    global x
    x = 1
10. TypeHint
def f(a: int) -> int:
    return a
